How It Works: The Core Logic

Dynamic Signal Optimization

The system calculates a 'Traffic Pressure Index' (TPI) for each lane by counting vehicles and assigning weights (e.g., Truck=3, Car=2). The lane with the highest TPI is allocated a proportionally longer green light, maximizing throughput.

Emergency Vehicle Green Corridor

When an emergency vehicle is detected, the STMS overrides the current cycle. It turns the vehicle's approach lane green and all other lanes red, creating a safe and clear path through the intersection without delay.
